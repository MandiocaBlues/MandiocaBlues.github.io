Scale Pot Info





Nobody can practice all scales everyday,

and noone should practice only the ones they like!


This webpage allows piano students & teachers

to pick random scales to practice,

from sets chosen by either ABRSM or TRINITY.


Place a shortcut to this webpage in your home screen.

It will keep a record of the practice.


Webpage written by M.A. Schmidt

while learning HTML, CSS & javascript.

There are no "cookies", no information is gathered.


Source code available in GitHub.

Code tested on Android only (chrome & sansung).
