Scale Pot Info





Nobody can practice all scales everyday,

and noone should practice only the ones they like!


This webpage allows piano students & teachers to pick random scales to practice,

from sets chosen by either ABRSM or TRINITY exam boards.

Place a shortcut to this webpage in your phone home screen.


The webpage keeps a record of the student's work in memory until it is reset.

There are no "cookies" and no information is gathered.


Webpage written by M. Schmidt while learning HTML, CSS & javascript.

Source code available in GitHub.


Code tested on Android only (chrome and sansung browsers).
